Project Overview

Boba-bot is a Slack chatbot that can answer any question by leveraging API.AI's machine learning capabilities, Cleverbot API, Google API's, and Wikipedia. API.AI classifies the questions into different categories. If it is a question like - Are there any restaurants nearby? - then Google Places API is used to answer the question. If it is a question like - Who is Alan Turing - then Wikipedia is scraped to formulate a good answer for the user. The responses for any other types of questions or casual talk is generated using Cleverbot's API. It is deployed using Heroku.

Skills Learned

Google API's, Slack chatbots, Machine Learning, Data Classification, Web Scraping, REST API's, Node.js, Heroku

Resources Used

Project Overview

Linkshare is a web application similar to Hacker News. Users can post links about latest things happening on the internet. Other users can comment on these posts or upvote them. The posts with the highest votes are displayed at the top. It has a full fledged login and registration system. It uses MongoDB for storing users, posts, comments, and upvotes. AngularJS was used to build the frontend, and Node.js and Express.js for the backend.

Skills Learned

Node.js, AngularJS, MongoDB, ExpressJS

VW emissions – I feel given the circumstances, the engineers at Volkswagen were faced with a very difficult choice – do whatever it takes to get the expected results or face the consequences. They had to do the impossible or else they would end up losing their jobs. For many, losing their jobs may not be something they could even consider.
